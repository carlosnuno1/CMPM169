Experiment 6 - Haiku Generator

Description

This project is a haiku poem generator using with it's wordbank being comprised of all the lyrics from the ablum Errorzone by Vein.fm.

Click for more poems.

Technical

Each word in the wordbank is given a syllable count and catorgirzed by type of word(noun, verb, adjective, etc.)

The the algorithm that choose the words for the poem has weights for type of words and and strict rules for syllable count to create more coherent poems while staying within the haiku structure.

It's far from perfect, but it helps to create a more intersting and undertsandble poem with some kind of message behind it.

Reflection

Overall I'm pretty happy with the end result of the project. Not the most technically impressive, or most creative project, but it was fun making it and a learned a solid bit from making it.
