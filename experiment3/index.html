Experiment 3 - Audio Visualizer

Description

Needed to make something generative but I didn't like the randoms and algorithm aspect of generative methods so I decided to make an audio visualizer.

Technical

Started by getting the mic input working and setting the volume to change the size of a circle. From there I made it so only three circles max could be on screen. I also added a button to toggle fullscreen mode.

I also made it so circles would disappear after a certain amount of time. From there I added a triangle and square. Was those were on the screen I made it so each shape had their frequency they were following

Triangle for highs, circle for mids, and square for lows. They are also all independt of each other for their size meaning the higher of its own frequency, the larger it would be without affecting the other shapes.

Reflection

Honestly I had lower expectations for this experiment that I was able to create. I've been thinking of this project for a while now I thought I would be limited more by js but it turns out I was able to make something I think it pretty cool.

There are still some shortcomings with the project especially the fact that their are limits to audio reactive stuff in terms of input devices and what a user wants to input. Some inputs may not produce something all that interesting.
